# Task-Manegar 📝

A simple task manager built with Node.js, Express, MongoDB, and React.

## ✨ Features

- Add, update, and delete tasks  
- Task filtering and organization  
- Clean, responsive UI  
- RESTful API backend

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- MongoDB (local or cloud)
